about

this website

This is my personal website, which serves as a portfolio, knowledge repository, and experiment log.

It is meant to be a living document whose pages get edited and moved around, while preserving their URL for reliable bookmarking.

Like most of what I make, it also serves as an exercise for me to learn something new (HTML and CSS in this case), which means some things aren't quite finished yet.

technicalities

This website is generated by a haphazardly written Python script, which takes Markdown files written in Obsidian and converts them into HTML files.

The images are optimized for size with ImageMagick, using tips from Dook. GIFs are converted into smaller MP4 files with an FFmpeg script stolen from rostiger's website.

The fonts I use for the logo and headings is Fredoka. For the text body I use Josefin.
