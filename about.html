about

this website

This place serves as a personal portfolio, knowledge repository, and experiment log. It is a living document whose pages get edited and moved around.

Like most things I make, this website is an exercise for me to learn something new (namely HTML and CSS). So there might be a few wonky things here and there, but hey, it works.

technicalities

This website is generated with potoo, a haphazardly written Python script that takes Markdown files written in Obsidian and converts them into HTML files.

Images are optimized for size with ImageMagick, using tips from Dook, and GIFs are converted into smaller MP4 files with an FFmpeg script stolen from rostiger's website.

The fonts I use are Fredoka for the headings, and Josefin Sans for the text body.

me

I'm a maker/hacker/tinkerer from France who goes by ghettobastler online. My pronouns are he/him. I like to learn new things and I build my own tools to achieve that.

You can find me on the Fediverse to see what I'm up to.
