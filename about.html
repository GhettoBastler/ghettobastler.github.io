about

This website

I consider this to be my personal digital garden and, as such, things might move around and get edited. I was inspired by other personal websites from the webring.

Like most things I make, this site was built to learn something new (namely HTML and CSS), therefore there are a few wonky things here and there, but hey, it works.

Technicalities

These pages start as Markdown notes written with, and are converted into HTML by a haphazardly written Python script called potoo. Images are optimized for size with ImageMagick, using tips from Dook, and GIFs are converted into smaller MP4 files with an FFmpeg script stolen from rostiger.

The fonts I use are Fredoka for the headings, and Josefin Sans for the text body.

This website uses GoatCounter to count how many people view its pages. It doesn't store personal information about visitors, nor does it uses client-side data storage (cookies or other methods).
